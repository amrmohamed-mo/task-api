Task API



Simple REST API built using Python Flask and Docker.



Technologies Used



\- Python

\- Flask

\- Docker

\- Git

\- GitHub



\---



Project Features



\- Simple Flask API

\- Returns task data in JSON format

\- Dockerized application

\- Version controlled using Git

\- Uploaded to GitHub



\---



Run Locally



python app.py



\---



Build Docker Image



docker build -t task-api .



\---



Run Docker Container



docker run -p 5000:5000 task-api

/
